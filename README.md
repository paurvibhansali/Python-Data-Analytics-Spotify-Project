This project combines Python-driven exploratory data analysis with concepts from Behavioral Economics to examine the evolution of popular music between 1950 and 2024. Through the use of Pandas, NumPy, Matplotlib, and Seaborn, the study analyzes changes in key musical attributes and investigates how the rise of streaming platforms may have influenced music production. Using the Spotify track dataset, the research explores whether songs have become shorter, more energetic, less acoustic, and increasingly engagement-oriented as competition for consumer attention has intensified over time.
